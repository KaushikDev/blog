Comparison Operators

Introduction

A comparison operator compares its operands and returns the logical value based on whether the comparison is true. Generally, the operands are of same type and if not, Javscript tries to convert them to same type.A comparison operator can compare the Numerical values, Strings, Objects or logical values.

Types of Comparison Operators

There are following types of Comparison operators :

Equal(==) //Return true if operands are equal Not equal(!=) //Return true if operands are not equal Strict equal(===) //Return true if operands are equal and of the same type Strict not equal(!==) //Return true if operands are equal but of different type or are not equal butof same type. Greater than(>) //Return true if left operand is greater than the right operand Greater than or equal(>=) //Return true if left operand is greater than or equal to the right operand Less than(<) //Return true if left operand is less than the right operand Less than or equal(<=) //Return true if left operand is less than or equal to the right operand

Let's code!

Let's see each type of comparison operator in action. Open the Chrome browser console window and type the following:

let variableOne = 5; let variableTwo = 10; console.log(5 == variableOne); //Will return true console.log("5" == variableOne); //Will return true console.log(5 == '5'); //Will return true console.log(5 != variableTwo); //Will return true console.log(10 != variableOne); //Will return true console.log(variableTwo != 5); //Will return true console.log(variableOne != "10"); //Will return true console.log(5 === variableOne); //Will return true console.log('5' !== 5); //Will return true console.log(variableOne !== '5'); //Will return true console.log(variableTwo > variableOne); //Will return true console.log("6" > variableOne); //Will return true console.log("6" >= variableOne); //Will return true console.log(variableTwo >= 8); //Will return true console.log(variableTwo >= variableOne); //Will return true console.log(variableOne < variableTwo); //Will return true console.log("6" < variableTwo); //Will return true console.log("6" <= variableTwo); //Will return true console.log(variableOne <= 8); //Will return true console.log(variableOne <= variableTwo); //Will return true

Output

Now hit enter and see if you get the desired output. Observe how sometimes strings and numbers were compared and still you got the output. That's where type conversion was done. Type conversion does not happen however with strict equality operator(===) and strict inequality operator(!==) and these perform strict equality and inequality comparisons.

By now, we have a good understanding of Comparison Operators. Comparison operators are life of javscript programming and are used more often than not. Hope you enjoyed this blog. Until next time. Happy coding.
