Using Variables

Introduction

Variables are an integral part of any programming/scripting language. Javascript is no exception. They give you flexibility of reusing the same 'data' over aand over again and also change 'data' whenever deemed necessary. In this blog we will see how to use variables.

Back to Chrome's console

In Chrome's console window, type the following :

let myName = "kaushikDev";
console.log("My Name is : "+myName);

Output

As soon as you hit enter key you'll see that console window has generated an output as "My Name is : kaushikDev". Awesome isn't it? So what happened here? We declared a variable "myName" and defined it with "kaushikDev". In simple terms, our variable holds the value "kaushikDev". So now whenever we want to use the value "kaushikDev" we can just call our variable, like we did in the console.log instruction.

What's that 'let' thing?

You must be wondering what this 'let' is? It is a javascript keyword(keywords are special words that are meant for specific use.). It lets javascript know that we want to declare a variable which we might be using in the program.Let's see another example. Type the following in the console window :

let firstName = "Piyush";
let lastName="Kaushik";
console.log("My Name is : " + firstName + " " + lastName);

Conclusion

Hit enter key and see what happens. We got "My Name is : Piyush Kaushik" as the output. Here, we used two variables 'firstName' and 'lastName' and used those in console.log instruction. The '+' operator just combines(concatenates) the two strings. So by now you have a brief idea on what variables are and how to use them, In coming blogs we will see multiple examples of variables, strings, operators and many more. Happy Coding.
