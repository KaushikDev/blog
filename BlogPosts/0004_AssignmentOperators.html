Assignment Operators

Introduction

An assignment operator is used to assign value to a variable. The value to the right side of the assignment operator is assigned to the variable on the left side. Assignment operator is denoted by single '=' sign. Few examples of assignment operation are :

let y = 5; //assigning 5 to variable y let x = y; //assigning the value of variable y to variable x let z = 5 * 2 + 9; // assigning the value of expression to variable z const myName = "kaushikDev"; // assigning a string "kaushikDev" to variable 'myName' let myCollection = ["Star Wars Memorabilia", 5]; //assigning an array to variable 'myCollection'

Compound Assignment Operators

Compound Assignment Operators are generally the combination of two types of operators of which one is assignment operator and other is generally(there are others as well but most frequently used are arithmetic ones.) an arithmetic operator. Examples of Compound Assignment Operators are :

x += y //which literally means x = x + y x -= y //which literally means x = x - y x *= y //which literally means x = x * y x /= y //which literally means x = x / y x **= y //which literally means x = x ** y

Ok, let's code now!

Let's code a simple javascript program. Open the Chrome browser console window and type the following:

let y = 5; console.log("The value of y is : " + y); let x = y; console.log("The value of x is : " + x); y += y; console.log("The value of y is : " + y); x -= 3; console.log("The value of x is : " + x); x *= y; console.log("The value of x is : " + x); x /= y; console.log("The value of x is : " + x); x **= x console.log("The value of x is : " + x);

Output

Hit enter key and observe the output. It looks like :

The value of y is : 5 //because we assigned integer 5 to variable 'y'.
The value of x is : 5 //because we assigned value of variable 'y' to variable 'x'.
The value of y is : 10 //because we assigned value of variable 'y' to variable 'y' i.e. y = y + y = 5 + 5 = 10.
The value of x is : 2 //because we subtracted 3 from the value of varibale 'x' i.e. x = x - 3 = 5 - 5 = 2 and assigned to x.
The value of x is : 20//because we multiplied values of variables 'x' and 'y' and assigned it to variable 'x'.
The value of x is : 2//because we divided value of variable 'x' by value of variable 'y' ans assigned it to variable 'x'.
The value of x is : 4//because we 'exponentiated' the value of variable 'x' and assigned it to variable 'x'.

Conclusion

I am confident that by now you have a fair understanding of assignment operators. Try to play with it and observe what you get. Hope this was somewhat helpful to you. Happy Coding.
