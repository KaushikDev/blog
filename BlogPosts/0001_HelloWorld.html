Introduction

Hello World! In this blog we will learn how to write our first program in javascript. We could use an IDE(Integrated Development Environment) too like "Visual Studio Code" to write out first program, but for the sake of simplicity we are going to use Chrome's console. So let's begin.

Navigating to Chrome's console

Open the Chrome browser on your desktop/mac and right click and select 'Inspect'. You could use the function key f12 to do the same. Navigate to 'Console' and there you are. Now we'll begin typing our very first program.

Coding First Program

In console window, type the following :

console.log("Hello World!");

As soon as you hit enter key, you will see the output generated as "Hello World". There you have it. Your very first javascript program. It's a great way to start. Try out with different strings. (A piece of text is generaly called a string in javascript.)

What's next?

Next few blogs will concentrate on exploring javascript basics and web development in general. We will write code along the way. This is to make sure that whoever is complete beginner or has never coded before would feel comfortable in coding. Happy Coding.
