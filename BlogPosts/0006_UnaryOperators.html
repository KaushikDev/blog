Unary Operators

Introduction

A Unary Operator is an operator that works with a single operand or in other words, a unary operation is performed with just one operand. Most commonly used Unary operators are, Unary plus(+), Unary negation(-), Logical not(!), typeof, delete, void, Bitwise NOT operator(~). Here in this blog, we will discuss some of the important ones.

Unary plus(+)

The Unary plus operator precedes its oprand and tries to convert it into a number if it isn't already. It is the fastest way to convert something into a number. If it cannot evaluate something it returns NaN. Let's see it in action in your browser's console window.

+3 // return 3 +'3' // return 3 +true // return 1 +false // return 0 +null // return 0 +"kaushikDev" // return NaN +function(val){return val} // return NaN

Unary negation(-)

The Unary negation operator precedes it's operand and negates it. Let's see it in action.

-3 // return -3 -'3' // return -3 -true // return -1 -false // return -0 -null // return -0 -"kaushikDev" // return NaN

Side note: Truthy and Falsy.

If a value can be converted to true, it is considered as truthy and if a value can be converted to false, it is considered as falsy.

Logical not(!)

It returns false if single operand can be converted to true, otherwise returns true. Let's see the exmaples below:

!true // returns false !false // returns true !'' // returns true !"kaushikDev" // returns false !null // returns true !NaN // returns true !0 // returns true !1 // returns false !(1 > 0 || 8 > 0) // returns false

typeof

The typeof operator returns a string indicating the type of the unevaluated operand. The syntax is :

typeof operand typeof(operand)

Let's see some examples.

typeof 37 // returns "number" typeof '42' // returns "string" typeof(9) // returns "number" typeof Math.PI // returns "number" typeof "kaushikDev" // returns "string" typeof {test: true} // returns "object" typeof [34, "developer"] // returns "object" typeof null // returns "object" typeof NaN // returns "number" typeof undefined // returns "undefined" typeof true // returns "boolean" typeof Date // returns "function"

delete

Though we have not yet covered Objects, but just keep this is in mind that delete operator removes a property of an object. Consider the following example :

const Employee = { firstname: 'John', lastname: 'Doe' }; // this is how you declare an object. console.log(Employee.firstname); // expected output: "John" delete Employee.firstname; console.log(Employee.firstname); // expected output: undefined

void

The void operator evaluates the given expression and then returns undefined. If interested, you can read more about it here.

Conclusion

Now we have a good understanding of Unary Operators. We will be using these a lot in future examples and blog posts. Like always try these on your own and observe the ouputs. Until next time. Happy coding.
